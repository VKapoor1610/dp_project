Letter Practice Mode Data Analysis

Correct and Incorrect Questions by Date

This bar plot visualizes the performance over time, showing the number of correct and incorrect questions answered on each date.

The black line shows the total number of questions answered on each date.

Recorded accuracy Over Time

This plot shows the accuracy of the child's answers over time.

The accuracy is calculated as the percentage of correct answers out of the total number of questions answered on each day.

The black line represents the accuracy trend, and the corresponding dots represent the accuracy on each day.

Percentage Letter Completion

This horizontal bar plot represents the how much is the child far away from completion of given letter.

The plot provides insight into which letters need more focus on and our adaptive learning model will make sure of teaching such letters to teach the child.
